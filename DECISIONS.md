# DECISIONS

1. **Product: CSV→QBO/OFX/QIF converter ("Ledgerport").** Reason: QuickBooks Desktop cannot import CSV; paid competitors prove willingness to pay (MoneyThumb $25/mo, DocuClipper $39/mo, ProperSoft ~$60 one-time); free web competitors upload financial data to servers — local-first privacy is the differentiator. Rejected: bank-reconciliation diff tool (weaker discoverability), PDF statement parser (error-prone, not finishable to quality). Consequence: fully deterministic core = fully unit-testable.
2. **Stack: Vite + React + TypeScript, PapaParse, Vitest, hand-written CSS.** Reason: fast to finish, static output deploys anywhere free, strong typing for the format generators. Rejected: Next.js (needless server), Tailwind (extra dep for one page).
3. **No accounts, no backend, no analytics.** Reason: core value needs none; privacy is the selling point. Consequence: presets persist in localStorage only.
4. **QBO = OFX 1.02 SGML + INTU.BID; OFX = 2.1.1 XML; QIF per Quicken spec.** NAME capped at 32 chars (overflow → MEMO), FITID = date+amount+seq hash, credit-card accounts use CCSTMTRS/CCACCTFROM. Known honest limitation: QuickBooks validates INTU.BID against its bank directory — user may need their own bank's ID; documented in-app with common IDs.
5. **Deployment: static `dist/` + netlify.toml + vercel.json; live single-file build published as a claude.ai Artifact.** Reason: no platform credentials exist in this environment (verified); Artifact is the only live-hosting path available without owner login. Public deploy = 1 human drag-drop step.
6. **Amount conventions supported: single signed column, debit/credit twin columns, sign-flip toggle, parentheses/`$`/commas/trailing-minus/European `1.234,56`.** Reason: these are the exact failure modes bank CSVs exhibit; competitors' handling of these is the main complaint.
7. **Pricing hypothesis: free during launch; Pro $29/yr (mapping presets sync, batch multi-file, priority support) once demand is proven.** Undercuts MoneyThumb/DocuClipper subscriptions; one-time-feel annual price matches ProperSoft buyers. No checkout built (no payment account authorized).
